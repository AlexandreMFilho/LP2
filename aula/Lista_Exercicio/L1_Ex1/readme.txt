Esse programa calcula a area de 3 tipos de figura; circulos, quadrados e triangulos (e tambem informa que tipo de triangulo ele é ,isosceles, escaleno ou equilatero).
Ele ao ser executado deve receber de 1 a 3 argumentos para execucao do programa. Com o uso do polimorfismo do java o programa diferencia a figura pelo numero de dados informados, caso nenhum dado seja passado o programa irá informar que não foi passado nenhum argumento, alem disso caso seja passado argumentos demais ele tambem o informara o ocorrido e caso o argumento nao obedeca o formato necessario o programa tambem ira informar.
Por favor insira os argumentos na execucao do programa e cuide para que os dados fornecidos sejam do tipo numerico.
